# Patient Treatment-Recovery Data Analysis in Excel
This project is carried out to analyze the treatment and recovery statistics of patients in a hospital. The KPIs were analyzed across departments, Physicians, Age group, Length of Stay(LOS), and in terms of cost 
## The objective is to:
- Identify drivers of recovery outcomes
- Compare physician and departmental performance
- Analyze cost vs outcome relationships
- Support hospital-level operational decision-making
- This project simulates a health system performance analytics tool suitable for quality improvement and value-based care models
